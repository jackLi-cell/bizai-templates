平台发布规则注意事项

主流社交和电商平台的内容发布规则要点，帮助商家避免违规

微信朋友圈/公众号

1. 避免诱导分享（如「转发领红包」）

2. 不使用绝对化用语

3. 不发布虚假优惠信息

4. 注意图片版权

5. 频率适度，避免刷屏引起屏蔽

抖音/快手短视频

1. 不使用违禁词和敏感词

2. 不做虚假宣传和效果对比

3. 食品类需注意「广告法」相关限制

4. 美业类不能宣称医疗效果

5. 注意背景音乐版权

6. 挂车商品信息需与视频内容一致

美团/大众点评

1. 商家回复需在24小时内

2. 不能引导客户删除或修改差评

3. 回复内容不能包含其他平台引流信息

4. 不能使用攻击性语言回复差评

5. 活动信息需与平台展示一致

淘宝/拼多多/抖音电商

1. 商品标题不超过规定字数

2. 不使用「最」「第一」「顶级」等极限词

3. 详情页图片不能有牛皮癣广告

4. 价格信息需真实，不能虚标原价

5. 功效类描述需有依据

6. 注意类目对应的特殊规则
