General Negative Review Response

Retail & E-commerce Customer Service Replies

Universal negative review response template for all industries

Use Cases

Use for responding to various platforms negative reviews , needing quick response

Not Suitable For

Not suitable for reviews involving legal disputes

Materials Needed

  • negative review platform
  • negative review content
  • issue reason
  • solution

Prompt (Click to Copy)

[Note: This prompt template is in Chinese, designed for use with Chinese-language AI models.]

你是一位客服经理。请针对以下差评撰写回复。

差评平台：{差评平台}
差评内容：{差评内容}
问题原因：{问题原因}
解决方案：{解决方案}

要求：
1. 真诚道歉
2. 承认问题不回避
3. 给出具体解决方案
4. 邀请私信沟通细节
5. 字数80-150字
6. 不与客户争论

Field Descriptions

FieldDescription
{差评平台}Dianping / Meituan / TikTok etc.
{差评内容}customer
{问题原因}verify reason
{解决方案}or

Output Format

80-150 character platform response

Output Example

[Sample output in Chinese - this is a format reference for Chinese-language AI output] 您好，看到您的反馈非常抱歉。您提到的等待时间过长问题确实是我们的不足，当天因为系统故障导致排队异常，目前已经修复。为表歉意，我们为您准备了一张8折优惠券，已私信发送给您。也欢迎您再次体验，我们会做得更好（仅供格式参考）

The above is a format example only. Actual output should be generated based on your materials.
