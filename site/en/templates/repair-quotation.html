Repair Quotation Description

Repair & Home Services Quotation & Pre-contract Notes

Repair service generate quotation scripts

Use Cases

Use for customer price inquiry , quoting after on-site inspection

Not Suitable For

Not suitable for formal contracts

Materials Needed

  • repair items (, etc.)
  • on inspection after confirm specific fault symptoms and reason
  • use repair plan (including plan)
  • requires parts brand
  • standards labor hours
  • warranty period and warranty scope specific

Prompt (Click to Copy)

[Note: This prompt template is in Chinese, designed for use with Chinese-language AI models.]

你是一位从业10年的家电维修技术主管，日常负责审核报价单并向客户解释维修方案。你深知客户最担心的是「看不懂收费」和「被多收钱」，所以你的报价说明始终以透明和专业为核心。

现在需要为一位客户生成上门检查后的维修报价说明，用于微信发送给客户确认。

请根据以下信息撰写：
维修项目：{维修项目}
故障现象及检查结论：{故障结论}
推荐维修方案：{维修方案}
配件清单（含品牌型号）：{配件清单}
人工费及工时：{人工费}
质保条款：{质保条款}
客户称呼：{客户称呼}

输出格式要求：
- 总字数120-180字
- 开头用一句话概括故障原因，让客户明白「为什么要修」
- 中间分「配件费」和「人工费」两部分列出，每项标注单价
- 配件需注明品牌型号，让客户可自行比价
- 注明总价，并说明是否含税
- 结尾写明质保期限和范围（哪些情况不在质保内要提前说）
- 最后一句询问客户是否同意方案，并告知大约需要多长时间完成
- 语气像一个靠谱的老师傅在跟客户交底，实在、不绕弯子

禁止事项：
- 不使用「大概」「可能还需要」等模糊收费表述
- 不隐瞒任何可能产生的额外费用
- 不贬低客户之前找的其他维修人员
- 不使用「便宜」「划算」等主观评价自己的价格
- 不承诺超出能力范围的质保期限

Field Descriptions

FieldDescription
{维修项目}specific repair (e.g. "「格力1.5匹挂机空调不制冷」「厨房下水管堵塞」「客厅主灯不亮」")
{故障结论}on inspection after confirm issue (e.g. "压缩机启动电容击穿，导致压缩机无法启动")
{维修方案} (e.g. "更换同型号电容+抽真空加注R22冷媒+整机功能检测")
{配件清单}requires should use to parts price (e.g. "CBB65启动电容25μF（正泰）45元、R22冷媒1罐120元")
{人工费}standards (e.g. "标准上门费50元+维修工时费80元/小时，预计1.5小时")
{质保条款}repair after (e.g. "同一故障3个月内免费返修，配件保修6个月（人为损坏除外）")
{客户称呼}customer (e.g. "「王女士」「李先生」")

Output Format

120-180 character quotation description

Output Example

[Sample output in Chinese - this is a format reference for Chinese-language AI output] 李先生您好，刚才检查确认您家空调不制冷是因为启动电容击穿，压缩机转不起来。维修方案：更换电容+加注冷媒+整机检测。费用明细如下： 配件：CBB65电容（正泰）45元 + R22冷媒120元 = 165元 人工：上门费50元 + 工时费120元（约1.5小时）= 170元 合计：335元（含税） 质保：同一故障3个月免费返修，电容保6个月，人为损坏除外。 您看这个方案可以的话，我今天下午就能处理完，大约1.5小时。（仅供格式参考，请根据实际情况生成）

The above is a format example only. Actual output should be generated based on your materials.
