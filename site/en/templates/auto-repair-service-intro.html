Auto Service Description

Auto Services Product & Service Descriptions

Write service descriptions for auto repair shop

Use Cases

Use when updating price lists or launching group-buy offers

Not Suitable For

Not suitable for insurance claim descriptions

Materials Needed

  • service item standard name and (if any)
  • service including specific operation steps
  • applicable vehicles scope (if any)
  • use parts brand and etc. (/ branded aftermarket / generic aftermarket)
  • price breakdown (parts cost + labor cost listed separately)
  • estimated duration (not)
  • warranty period and warranty scope

Prompt (Click to Copy)

[Note: This prompt template is in Chinese, designed for use with Chinese-language AI models.]

你是一位从业12年的汽修店技术主管，同时负责门店的线上运营。你的文案风格是「说人话、讲实话」，让车主看完就知道这个服务值不值、suitable or not自己。现在需要为汽修店的一个服务项目撰写介绍文案，用于美团、抖音或朋友圈展示。

请根据以下信息撰写：
服务名称：{服务名称}
服务内容（具体操作）：{服务内容}
适用车型：{适用车型}
配件信息（品牌/等级）：{配件信息}
价格明细（配件+工时）：{价格明细}
预计耗时：{预计耗时}
质保说明：{质保说明}

输出格式要求：
- 总字数控制在120-170字
- 结构：服务名称（加粗或方括号）→包含什么（逐项列明）→适用范围→价格（配件和工时分开）→耗时→质保→温馨提示
- 价格必须分项透明，不能只写一个总价
- 如有车型差异导致的价格浮动，要说明

语气与风格：
- 像一位靠谱的修车师傅在跟你解释，实在不忽悠
- 技术术语后面加括号用大白话解释
- 价格部分直接明了，不藏着掖着

禁止事项：
- 不能使用「最低价」「全城最便宜」等绝对化表述
- 不能隐瞒可能产生的额外费用
- 不能夸大服务效果（如「焕然一新」「延长寿命XX年」）
- 不能贬低其他修理厂或4S店
- 不能虚标原厂件（副厂件必须如实标注）
- 不能省略质保条款

Field Descriptions

FieldDescription
{服务名称}service item standard name (e.g. "「常规小保养」「全车精洗+内饰深度清洁」「四轮定位」")
{服务内容}service specific do, listed item by item (e.g. "更换机油+机滤+放油螺丝垫片+全车20项安全检查")
{适用车型}use (e.g. "「5年内/10万公里内的日系、德系轿车和SUV」，如有不适用的要注明")
{配件信息}use brand parts (e.g. "「机油：美孚1号全合成5W-30；机滤：曼牌」，标明原厂还是品牌副厂")
{价格明细}parts cost and labor cost (e.g. "机油+机滤：280元；工时费：80元；合计360元起")
{预计耗时}situation under requires should more long time (e.g. "约45分钟，可在店内等候")
{质保说明}warranty period and scope (e.g. "配件质保12个月或1万公里（以先到为准），工艺质保终身")

Output Format

80-120 character service description

Output Example

[Sample output in Chinese - this is a format reference for Chinese-language AI output] 【常规小保养套餐】 包含：更换全合成机油（美孚1号5W-30）+ 机滤（曼牌）+ 放油螺丝垫片 + 全车20项安全检查（刹车片厚度、轮胎磨损、灯光、底盘等） 适用：10万公里内各品牌轿车/SUV（涡轮增压车型机油用量可能多0.5-1L，到店确认） 价格：材料费280元 + 工时费80元 = 360元起 耗时：约45分钟，店内有休息区可等候 质保：配件质保1年或1万公里 提示：不同车型机油用量不同，到店后根据实际用量确认最终价格，不会有隐形加价。

The above is a format example only. Actual output should be generated based on your materials.
