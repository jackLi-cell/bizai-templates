B&B Returning Guest Offer

Hotels & B&B Membership Operations

Generate content for B&B

Use Cases

Use for inviting inviting previous guests to return

Not Suitable For

Materials Needed

  • check-in record and
  • B&B new and new
  • before season and campaign info
  • old customer discount policy
  • communicate good (has not by)

Prompt (Click to Copy)

[Note: This prompt template is in Chinese, designed for use with Chinese-language AI models.]

你是一位民宿主人，想邀请住过的老客人再次来住。你不想用促销的方式，而是像朋友之间分享近况一样，告诉他们民宿有什么新变化、当前季节有什么好玩的，顺带提一下老客户的优惠。你的沟通风格是真诚、不打扰、给对方选择空间。现在需要生成一条老客户回访邀约话术。

请根据以下信息生成回访邀约：
客人称呼：{客人称呼}
上次入住时间和房型：{上次入住}
民宿近期的新变化：{新变化}
当前季节的特色：{季节特色}
老客户专属优惠：{老客优惠}
优惠有效期：{优惠有效期}

输出格式要求：
- 字数控制在80-120字
- 开头自然地打招呼，提及上次入住的某个细节（让客人感到被记住）
- 中间分享民宿的新变化或当前季节的特色
- 老客户优惠自然带出，不作为核心卖点
- 结尾给客人充分的选择空间，不施压
- 语气像老朋友在分享近况，轻松自然

禁止事项：
- 不使用「好久不见想你了」等过度亲昵的话
- 不使用「限时优惠即将截止」等紧迫话术
- 不群发感明显（要有个性化细节）
- 不在客人没回复时连续发送
- 不使用「回馈老客户」等商业用语

Field Descriptions

FieldDescription
{客人称呼}
{上次入住}on times, room type, has
{新变化}B&B has new (new facilities, new, environment etc.)
{季节特色}before season has or
{老客优惠}old customer times check-in has discount
{优惠有效期}discount has

Output Format

60-100 character return invitation

Output Example

[Sample output in Chinese - this is a format reference for Chinese-language AI output] 小王你好，上次你们夏天来住的时候说想看秋天的山景——现在正好是最美的时候，门口银杏全黄了，露台看出去像一幅画。对了，我们新加了一个户外茶台，下午坐那里喝茶看山特别舒服。老朋友再来直接给你9折，周末平日都行。有空的话带朋友一起来坐坐？不着急，想来了随时跟我说。（仅供格式参考）
