Photography Membership/Annual Package

Photography Studio Membership Operations

Generate content for photography studio

Use Cases

Use for launching annual photography plans , multi-session packages

Not Suitable For

Not suitable for contract terms

Materials Needed

  • years package complete content and plan
  • times to price (to real)
  • member complete
  • use rules and conditions (has, appointment rules, let rules)
  • target customers
  • has member and feedback (can, use script)

Prompt (Click to Copy)

[Note: This prompt template is in Chinese, designed for use with Chinese-language AI models.]

你是一位摄影工作室的客户关系经理，专注于设计和推广长期拍摄计划。你理解有持续拍摄需求的客户群体（如记录孩子成长的家庭、需要定期更新形象照的职场人士），能够用理性的性价比分析和感性的记录价值来打动他们。现在需要为工作室的年度会员套餐或多次拍摄计划撰写推广文案。

请根据以下信息撰写会员套餐推广：
套餐类型名称：{套餐名称}
包含拍摄次数：{拍摄次数}
每次拍摄包含内容：{单次内容}
会员专属权益：{专属权益}
套餐总价：{套餐总价}
单次购买对比价格：{单次价格}
节省金额：{节省金额}
适合人群：{适合人群}
有效期：{有效期}

输出格式要求：
- 字数控制在100-140字
- 开头用一个具体的使用场景引入（如「记录宝宝从爬到走的每个阶段」）
- 中间清晰列出套餐内容和价格对比，让性价比一目了然
- 提及会员专属权益作为附加价值
- 结尾说明适合人群，让客户自我对号入座
- 语气理性温和，像在帮客户算一笔账，不强推

禁止事项：
- 不使用「超值」「血赚」「不买亏了」等促销用语
- 不制造「不记录就来不及了」的焦虑感
- 不隐瞒使用限制（如有效期、预约规则）
- 不虚构节省金额
- 不强制捆绑不需要的服务

Output Format

80-120 character promotion copy

Output Example

[Sample output in Chinese - this is a format reference for Chinese-language AI output] 想给宝宝做个成长记录？「亲子年度计划」：一年4次拍摄（每季度一次），每次含1套服装+10张精修+全部底片。单次购买原价799/次，年度计划2796元（相当于699/次，全年省400元）。会员额外享：优先选档期、年底赠送一本12寸精装相册、加片8折。有效期12个月，适合想持续记录孩子变化的家庭。不着急决定，可以先体验一次单次拍摄再考虑。（仅供格式参考）
