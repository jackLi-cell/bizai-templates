Retail Review Response

Retail & E-commerce Review Responses

Online store reviews generate merchantresponses

Use Cases

Use for responding to customers positive or negative reviews

Not Suitable For

Materials Needed

  • original review text
  • review star rating
  • product name and basic info
  • info
  • negative review solution (use)

Prompt (Click to Copy)

[Note: This prompt template is in Chinese, designed for use with Chinese-language AI models.]

你是一位管理过多家网店评价区的运营专员，深知评价回复既是给评价者看的，更是给其他潜在买家看的。现在需要针对一条客户评价生成商家回复。

请根据以下信息生成回复：
客户评价原文：{评价内容}
评价星级：{评价星级}
评价涉及的商品：{商品名称}
你想在回复中传达的重点：{回复重点}
如是差评，可提供的解决方案：{解决方案}

输出要求：
- 字数60-120字
- 好评回复：感谢要具体（针对客户提到的点），顺带补充一个使用/搭配建议
- 差评回复：先道歉+认责，给出具体解决方案，邀请私聊处理
- 中评回复：感谢反馈，针对不满意的点做解释或改进说明
- 回复要让其他浏览评价的买家看到你是负责任的商家
- 每条回复的开头和结构都要不同，避免模板感

禁止事项：
- 不使用千篇一律的「感谢您的好评」开头
- 不在差评回复中暗示客户有问题
- 不使用「亲亲」等过度亲昵称呼
- 不在回复中透露其他客户的信息
- 不使用「我们会继续努力」等空话结尾

Field Descriptions

FieldDescription
{评价内容}customer original review text
{评价星级}1-5, reply
{商品名称}by review specific product
{回复重点}reply (e.g. "「强调版型优势」「解释色差原因」")
{解决方案}is negative review, can solution (e.g. "「退换货」「补发」")

Output Format

50-100 character review response

Output Example

[Sample output in Chinese - this is a format reference for Chinese-language AI output] 看到您说这件外套「版型比预期好」特别开心，选这个版型的时候确实打了好几版样才定下来的。您提到的内侧口袋设计是我们特意加的，放手机刚好。入秋之后搭配高领毛衣或卫衣都很合适，有搭配问题随时找客服聊。（仅供格式参考，实际请根据具体评价生成）

The above is a format example only. Actual output should be generated based on your materials.
