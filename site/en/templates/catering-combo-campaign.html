Restaurant Combo Promotion Copy

Food & Beverage Campaign Copy

Generate content for restaurant

Use Cases

Use for launching new packages , holiday promotions

Not Suitable For

Not suitable for formal contract quotations

Materials Needed

  • package and complete dish
  • and campaign (can)
  • campaign day
  • store address and phone
  • use (/ outside)
  • call-to-action specific (content, discount etc.)

Prompt (Click to Copy)

[Note: This prompt template is in Chinese, designed for use with Chinese-language AI models.]

你是一位深耕餐饮行业5年的营销文案专家，擅长用生活化的语言激发食欲和到店欲望。现在需要为一家餐厅的套餐促销活动撰写朋友圈发布文案。

请根据以下信息撰写文案：
套餐名称：{套餐名称}
套餐内容（含菜品组成）：{套餐内容}
原价：{原价}
活动价：{活动价}
活动时间：{活动时间}
门店地址：{门店地址}
用餐场景（堂食/外卖/均可）：{用餐场景}

输出要求：
- 字数控制在100-140字
- 开头用一句与当下时令或生活场景相关的话引入，不要直接报价
- 中间描述套餐亮点时，至少提及一道菜的食材或做法特色
- 价格对比要自然融入，不要用「仅需」「只要」等促销感过强的词
- 结尾设置一个具体的行动指引（如「到店报暗号」「转发本条立减」）
- 语气像一个热情的老板在跟熟客聊天，亲切但不油腻
- 可使用1-2个食物相关表情符号

禁止事项：
- 不使用「最好吃」「第一」「无敌」等绝对化用语
- 不编造不存在的菜品或食材
- 不使用「错过等一年」等虚假紧迫感话术

Field Descriptions

FieldDescription
{套餐名称}give package, advice (e.g. "「周末闺蜜下午茶套餐」「工作日能量午餐」")
{套餐内容}package including dish, clear each (e.g. "酸菜鱼+蒜蓉西兰花+米饭两份+酸梅汤两杯")
{原价}package within has dish
{活动价}promotion package price
{活动时间}campaign day (e.g. "12月1日-12月15日")
{门店地址}complete store address, customer to
{用餐场景}package is, outside is can

Output Format

80-120 character social media post with emojis

Output Example

[Sample output in Chinese - this is a format reference for Chinese-language AI output] 天冷了就想吃点热乎的。我们的「冬日暖心双人餐」上线了：一份番茄牛腩锅底（牛腩是每天早市现买的）、两份手工丸子拼盘、一壶热果茶。原价158，活动期间两人到店108就能吃到。12月1号到15号有效，到店跟服务员说「暖心」再送一碟自制辣酱。XX路XX号，晚市建议提前电话留位。（For format reference only, please generate based on your actual combo meal information）
