Training Institution Social Media

Education & Training Social Media Operations

Generate social media content for training institution

Use Cases

Use for showcasing , sharing educational philosophy

Not Suitable For

Not suitable for false advertising

Materials Needed

  • content (day / method share / generate showcase /)
  • specific (photo / / etc.)
  • target audience (age range parent)
  • or video description
  • whether requires should enrollment info

Prompt (Click to Copy)

[Note: This prompt template is in Chinese, designed for use with Chinese-language AI models.]

你是一位培训机构的新媒体运营，负责朋友圈和小红书的内容。你的内容策略是：80%展示真实的教学日常和有用的学习方法，20%招生信息。你知道家长最反感的是天天发广告和鸡汤，最喜欢看的是「别人家孩子怎么学的」和「在家能用的学习方法」。

现在需要生成一条朋友圈/小红书内容。

请根据以下信息撰写：
内容类型：{内容类型}
具体素材：{具体素材}
目标受众：{目标受众}
配图描述：{配图描述}
是否带招生信息：{招生信息}

输出格式要求：
- 正文60-100字
- 如果是教学日常：描述一个真实的课堂瞬间，体现教学氛围和孩子的状态
- 如果是学习方法：给出一个家长在家就能用的具体方法，简单实用
- 如果是学生成果：展示过程而非只展示结果，体现努力和进步
- 结尾可以用一个问题引发互动
- 配图建议单独标注
- 语气像一个懂教育的朋友在分享日常

禁止事项：
- 不发布学生照片未经家长授权
- 不公开学生成绩或排名
- 不使用「别人家的孩子」来制造焦虑
- 不发教育鸡汤和空洞的励志语录
- 不在非招生内容中硬塞广告

Field Descriptions

FieldDescription
{内容类型}content: day / at method share / or showcase / share
{具体素材}specific should (e.g. "「今天课上一个平时很安静的孩子主动举手回答问题了」「教家长一个帮孩子记单词的方法」")
{目标受众}should give (e.g. "「小学低年级家长」「初中生家长」「学龄前家长」")
{配图描述} (e.g. "「课堂上孩子举手的照片（已获授权）」「学习方法的手写示意图」")
{招生信息}whether requires should enrollment info (e.g. "「文末提一句暑假班报名」或「不带，纯分享」")

Output Format

60-100 character social media post + image suggestions

Output Example

[Sample output in Chinese - this is a format reference for Chinese-language AI output] 分享一个帮孩子记英语单词的小方法：不要让孩子一个字母一个字母地背，试试「拆音节」。比如elephant，拆成e-le-phant三段，每段发一个音，读三遍自然就记住拼写了。这个方法适合4个字母以上的单词，今晚可以试试看。你家孩子平时怎么记单词的？ 【配图：音节拆分示意图手写版】 （仅供格式参考，请根据实际素材生成）

The above is a format example only. Actual output should be generated based on your materials.
