Retail New Arrival Promotion

Retail & E-commerce Campaign Copy

Newly arrived products write promotion copy

Use Cases

Not Suitable For

Materials Needed

  • new and quantity
  • (/ / can)
  • accurate price
  • channel and
  • on new discount policy (if any)
  • description

Prompt (Click to Copy)

[Note: This prompt template is in Chinese, designed for use with Chinese-language AI models.]

你是一位经营实体零售店的朋友圈运营高手，擅长用轻松的方式让老客户知道「店里来新货了」。现在需要为一批新到货的商品撰写朋友圈上新文案。

请根据以下信息撰写：
商品类别：{商品类别}
这批新品的共同亮点：{新品亮点}
价格区间：{价格区间}
到店/线上购买方式：{购买渠道}
上新优惠（如有）：{上新优惠}
适合什么样的人：{适合人群}

输出要求：
- 字数80-120字
- 开头用一个与季节、场景或需求相关的切入点，不要直接说「上新了」
- 描述新品时用一个具体的细节让人有画面感（颜色、手感、穿上的感觉）
- 价格信息自然带出，不要像价目表
- 如果有上新优惠，放在最后作为「顺便一提」的感觉
- 语气像在跟闺蜜/朋友分享好物

禁止事项：
- 不使用「亲们」「宝子们」等电商客服用语
- 不使用「手慢无」「抢疯了」等虚假紧迫感
- 不承诺「穿了显瘦」「用了变白」等效果
- 不贬低旧款或其他品牌
- 不使用绝对化用语

Field Descriptions

FieldDescription
{商品类别}new (e.g. "「秋季外套」「真丝衬衫」「运动鞋」")
{新品亮点}or (e.g. "「面料防风防水」「今年流行的廓形」")
{价格区间}new price scope (e.g. "199-359元")
{购买渠道}customer (e.g. "「到店试穿」「微信下单包邮」「小程序商城」")
{上新优惠}new discount (e.g. "「前三天9折」「买两件减30」，没有则留空")
{适合人群}new (e.g. "「通勤族」「喜欢简约风的」")

Output Format

60-100 character new product copy

Output Example

[Sample output in Chinese - this is a format reference for Chinese-language AI output] 天气一转凉就想找件能直接套上出门的外套。这批秋款到了几件我自己很喜欢的：面料是防风防水的软壳材质，摸起来有点像桃皮绒，不厚但挡风。有卡其、黑色、雾霾蓝三个色，199-359的价格。到店能直接试，手感和上身效果比图片好。这周五之前到店新品都可以享9折。（仅供格式参考，实际请根据你的商品信息生成）

The above is a format example only. Actual output should be generated based on your materials.
