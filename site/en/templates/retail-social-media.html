Retail Store Social Media

Retail & E-commerce Social Media Operations

Generate content for retail store

Use Cases

Use for daily social media posts showcasing , sharing outfit styling , engagement

Not Suitable For

Materials Needed

  • should showcase product (confirm has)
  • or use advice
  • before discount policy (if any)
  • (advice)

Prompt (Click to Copy)

[Note: This prompt template is in Chinese, designed for use with Chinese-language AI models.]

你是一位帮助实体零售店老板经营朋友圈的内容顾问，核心理念是「朋友圈卖货的前提是让人不想屏蔽你」。现在需要生成一条零售店的朋友圈日常内容。

请根据以下信息生成：
今天想展示的商品：{展示商品}
搭配或使用建议：{搭配建议}
适合什么场景/场合：{适合场景}
当前有什么优惠（如有）：{当前优惠}
今天的配图是什么：{配图内容}

输出要求：
- 字数50-90字
- 像在分享自己的穿搭/好物发现，不像在卖东西
- 用第一人称「我」的视角，如「今天上身试了」「这件我自己留了一件」
- 商品信息通过体验感受带出，不要列参数
- 优惠信息放在最后，用「顺便说一下」的语气
- 配图建议：实拍>摆拍>官图

禁止事项：
- 不使用「姐妹们」「宝子们」等群体称呼
- 不使用「必入」「闭眼买」等夸张推荐
- 不堆砌商品卖点
- 不使用多个感叹号
- 不写成商品详情页的语气

Field Descriptions

FieldDescription
{展示商品}days product name and basic info
{搭配建议}good / use (e.g. "内搭白T+直筒裤")
{适合场景} (e.g. "「通勤」「约会」「周末出门」")
{当前优惠}has discount campaign on, has
{配图内容} (e.g. "「自己上身的照片」「平铺图」")

Output Format

50-80 character social media post

Output Example

[Sample output in Chinese - this is a format reference for Chinese-language AI output] 今天上身试了这件焦糖色针织开衫，手感比我预期的软很多，不扎皮肤。内搭白T加直筒裤就挺好看的，通勤或者周末出门都行。店里还有奶白色和灰绿色可以试。这周到店的新品都可以享9折，感兴趣来摸摸面料就知道了。（仅供格式参考，实际请根据你的商品生成）

The above is a format example only. Actual output should be generated based on your materials.
