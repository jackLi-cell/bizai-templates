Restaurant Menu Description

Food & Beverage Product & Service Descriptions

Write dish description copy for menus or delivery platforms.

Use Cases

更新菜单、上线外卖平台时使用

Not Suitable For

Not Suitable For用于食品包装上的配料表

Materials Needed

  • 菜品名称和完整食材清单
  • 食材来源（如有特色产地）
  • 详细烹饪方法和关键参数
  • 准确的口味描述
  • 实际出品的份量/重量
  • 过敏原信息

Prompt (Click to Copy)

你是一位为多家餐厅撰写过菜单文案的美食文案师，擅长用简短的文字让人产生食欲。现在需要为一道菜品撰写外卖平台或纸质菜单上的描述文案。

请根据以下信息撰写：
菜品名称：{菜品名称}
主要食材及来源：{主要食材}
烹饪方式与时间：{烹饪方式}
口味特点：{口味特点}
份量/适合人数：{份量信息}
过敏原提示（如有）：{过敏原}

输出要求：
- 字数50-90字
- 第一句用食材或做法的亮点切入，不要重复菜名
- 至少包含一个具体的数字（时间、温度、重量等），增加可信度
- 口味描述用对比或层次感的方式表达，如「外酥里嫩」「先咸后甜」
- 最后一句可以是份量建议或食用建议
- 语气克制专业，像一本讲究的菜单

禁止事项：
- 不使用「最」「极致」「无与伦比」等夸张词
- 不编造食材产地
- 不承诺健康功效（如「养胃」「美容」）
- 不使用网络用语或表情符号
- 不写成广告文案的语气

Field Descriptions

FieldDescription
{菜品名称}菜品的正式名称，如「酸菜炖五花肉」
{主要食材}核心食材及来源，如「本地农场黑猪五花肉、自家腌制酸菜」
{烹饪方式}具体做法和关键时间，如「小火慢炖3小时」「280度烤制8分钟」
{口味特点}味道的层次和特征，如「酸香开胃，微辣收尾」
{份量信息}重量或适合人数，如「约600g，适合1-2人」
{过敏原}含有的常见过敏原，如「含花生」「含乳制品」，没有则写「无」

Output Format

40-80字菜品描述

Output Example

选用本地农场黑猪五花，切厚片与自家坛子酸菜同炖3小时。肉质软烂不腻，汤底酸香开胃，微辣收尾。每份约600g，配米饭适合1-2人。含大豆制品（酸菜），介意者请提前告知。（仅供格式参考，实际请根据你的菜品信息生成）

The above is a format example only. Actual output should be generated based on your materials.
