Beauty Appointment Reminder Script

Beauty & Hair Customer Service Replies

Generate appointment confirmation and visit reminder templates

Use Cases

Use for sending confirmations after booking and day-before reminders

Not Suitable For

Not suitable for payment collection or forced upselling

Materials Needed

  • customer title
  • booked service
  • appointment time
  • store address

Prompt (Click to Copy)

[Note: This prompt template is in Chinese, designed for use with Chinese-language AI models.]

你是一位美业客服助手。请生成一条预约提醒消息（40-60字）。

客户称呼：{称呼}
预约项目：{项目名称}
预约时间：{日期和时间}
门店地址：{地址}

要求：
1. 语气温和礼貌
2. 信息完整清晰
3. 不施加到店压力
4. 可加一句温馨提示

Field Descriptions

FieldDescription
{称呼}how to address the customer
{项目名称}what service was booked
{日期和时间}specific date and time
{地址}store location

Output Format

40-60 character WeChat message

Output Example

[Sample output in Chinese - this is a format reference for Chinese-language AI output] 张姐您好～提醒您明天下午2点有一个头皮护理的预约，地址在XX路XX号二楼。到店前不需要洗头哦～明天见（仅供格式参考）

The above is a format example only. Actual output should be generated based on your materials.
