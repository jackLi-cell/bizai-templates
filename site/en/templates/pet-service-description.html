Pet Service Description

Pet Services Product & Service Descriptions

Write service descriptions for pet store

Use Cases

Use when updating price lists or launching group-buy offers

Not Suitable For

Not suitable for veterinary diagnosis descriptions

Materials Needed

  • service item (/ / / etc.)
  • use and scope
  • service specific process and long
  • use product brand or
  • price (or service)
  • precautions and not use situation

Prompt (Click to Copy)

[Note: This prompt template is in Chinese, designed for use with Chinese-language AI models.]

你是一位有5年经验的宠物店店长，同时也是持证宠物美容师。你在撰写服务介绍时，始终站在宠物主人的角度思考：他们想知道「我的毛孩子会经历什么」「用什么产品安不安全」「大概要多久多少钱」。你的介绍要让主人放心把宠物交给你。

现在需要为店里的一项服务撰写介绍文案，用于美团/大众点评的服务详情页或店内价目表。

请根据以下信息撰写：
服务项目：{服务项目}
适用宠物：{适用宠物}
服务流程：{服务流程}
使用产品/工具：{产品工具}
价格：{价格}
服务时长：{服务时长}
注意事项：{注意事项}

输出格式要求：
- 总字数130-200字
- 开头一句话说明这项服务适合什么情况的宠物（如「毛发打结严重」「换季掉毛期」）
- 流程按步骤简述，让主人知道宠物会经历什么
- 产品/工具要提及品牌或特点，体现专业和安全
- 价格按体型或项目分档列出
- 注意事项要包含：哪些情况不建议做（如皮肤病期间）、需要提前准备什么
- 语气温暖专业，像一个爱动物的专业人士在介绍

禁止事项：
- 不承诺「绝对安全」「零风险」等绝对化表述
- 不替代兽医给出医疗建议
- 不贬低其他宠物店的服务
- 不使用「毛孩子秒变小公主」等过度拟人化营销语
- 不隐瞒可能的不适反应（如部分猫咪洗澡会应激）

Field Descriptions

FieldDescription
{服务项目}specific service (e.g. "「全身洗护套餐」「造型美容」「长期寄养」「基础服从训练」")
{适用宠物} (e.g. "「中小型犬（15kg以下）」「所有品种猫咪」")
{服务流程}from to to steps (e.g. "检查皮肤→梳毛去结→洗浴→吹干→修剪造型→清洁耳道")
{产品工具}use give / (e.g. "「雪貂洗毛液（低敏配方）」「日本多格漫吹水机」")
{价格}standards (e.g. "小型犬（5kg以下）128元，中型犬（5-15kg）168元")
{服务时长}large requires should more long time (e.g. "全程约1.5-2小时，视毛量和配合度")
{注意事项}situation not can do / requires should (e.g. "皮肤有红肿或伤口的暂不接待，请先就医")

Output Format

130-200 character service description

Output Example

[Sample output in Chinese - this is a format reference for Chinese-language AI output] 【全身洗护套餐】适合定期清洁、换季掉毛期的狗狗。 流程：到店先检查皮肤状况→全身梳理去除浮毛和小结→使用雪貂低敏洗毛液清洗（敏感肌可换用燕麦配方）→专业吹水机吹干至毛根→清洁耳道和剪指甲。全程约1.5小时。 价格：小型犬128元 / 中型犬168元 / 大型犬218元（严重打结需加收梳结费30-50元） 提醒：皮肤有红肿、伤口或刚打完疫苗一周内的狗狗暂不接待洗护，建议先咨询医生。首次到店的宝贝建议主人陪同10分钟帮助适应环境。（仅供格式参考，请根据实际服务生成）
