Restaurant Short Video Script

Food & Beverage Short Video Scripts

Generate content for restaurant

Use Cases

Use for daily posting store visit style short videos , new product showcase videos

Not Suitable For

Materials Needed

  • video
  • should showcase dish or
  • store
  • long
  • can use equipment and
  • platform (TikTok / / Xiaohongshu)

Prompt (Click to Copy)

[Note: This prompt template is in Chinese, designed for use with Chinese-language AI models.]

你是一位专注餐饮类内容的短视频编导，熟悉抖音和快手的流量机制，知道什么样的餐饮视频容易获得推荐。现在需要为一家餐厅撰写一条短视频的拍摄脚本。

请根据以下信息撰写脚本：
视频主题/选题：{视频主题}
要展示的菜品或场景：{拍摄菜品}
店铺特色或记忆点：{店铺特色}
目标时长：{时长要求}
拍摄条件（设备/人员）：{拍摄条件}

输出要求：
- 按「开场hook（前3秒）→ 主体内容 → 结尾引导」三段式结构
- 每个镜头写清楚：画面内容、景别（特写/中景/全景）、运镜方式、时长
- 标注字幕文案和口播文案（如需要）
- 给出背景音乐风格建议（不指定具体歌曲）
- 开场前3秒必须有一个「停留钩子」：悬念、冲突或视觉冲击
- 所有内容基于手机竖屏拍摄

禁止事项：
- 不承诺「发了就能火」等效果
- 不建议使用有版权风险的音乐
- 不设计需要专业设备才能完成的镜头
- 不包含虚假或夸大的内容
- 不抄袭其他账号的创意

Field Descriptions

FieldDescription
{视频主题}video (e.g. "「后厨揭秘」「招牌菜制作过程」「顾客反应」")
{拍摄菜品}should showcase dish or
{店铺特色}has (e.g. "「20年老店」「老板是退伍军人」「食材每天现采」")
{时长要求}video duration (e.g. "「15-30秒」「60秒以内」")

Output Format

Storyboard script with scene descriptions

Output Example

[Sample output in Chinese - this is a format reference for Chinese-language AI output] 【开场hook 0-3秒】景别：大特写。画面：滚烫的油浇在鱼片上，滋滋冒烟，花椒粒在油里跳动。字幕：「这一勺油，温度必须是230度」 【主体 3-20秒】景别：中景切特写。画面：厨师片鱼→腌制→摆盘→浇油的完整过程。口播：「鱼是每天早上鱼市拿的活鲈鱼，片好之后先用料酒和姜去腥...」 【结尾 20-25秒】景别：中景。画面：成品端上桌，顾客夹起第一筷。字幕：「想看完整做法评论区扣1 / XX路XX号」BGM风格：轻快中国风（仅供格式参考）

The above is a format example only. Actual output should be generated based on your materials.
