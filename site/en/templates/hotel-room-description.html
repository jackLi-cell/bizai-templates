B&B Room Description

Hotels & B&B Product & Service Descriptions

Write room type descriptions for B&B/hotel

Use Cases

Use for launching on OTA platforms , updating website

Not Suitable For

Not suitable for false room descriptions

Materials Needed

  • and
  • outside real photo (not)
  • facilities brand and info
  • before has price
  • time and policy
  • description (, situation)
  • specific content and time

Prompt (Click to Copy)

[Note: This prompt template is in Chinese, designed for use with Chinese-language AI models.]

你是一位民宿/精品酒店的OTA运营专家，有5年携程、美团、小红书平台的房型文案经验。你擅长用精准的描述让客人在预订前就能「看到」房间的样子，同时严格遵守不夸大实际条件的原则。现在需要为一个房型撰写介绍文案，用于OTA平台或官网展示。

请根据以下信息撰写房型介绍：
房型名称：{房型名称}
房间面积：{房间面积}
床型和尺寸：{床型信息}
窗外景观：{景观描述}
卫浴配置：{卫浴配置}
特色设施/亮点：{特色设施}
房间风格/设计理念：{设计风格}
价格区间（平日/周末/节假日）：{价格区间}
入住时间/退房时间：{入退时间}
可住人数：{可住人数}
早餐情况：{早餐信息}
周边环境简述：{周边环境}

输出格式要求：
- 字数控制在130-180字
- 第一句用房间最大的亮点开头（景观、设计或体验感）
- 中间按「空间→设施→体验」的逻辑展开
- 设施描述要具体到品牌或材质（如「金可儿床垫」「科勒卫浴」）
- 价格信息单独一行，区分平日和周末
- 入退时间和可住人数作为基础信息标注
- 语气像一位用心的民宿主在介绍自己最满意的房间

禁止事项：
- 不使用「奢华」「顶级」「五星级体验」等过度修饰
- 不虚构不存在的设施或景观
- 不使用「仿佛置身XX」等虚幻描述
- 不隐瞒房间的实际限制（如无电梯、隔音一般等）
- 不使用其他酒店的照片描述来误导

Output Format

100-150 character room type description

Output Example
