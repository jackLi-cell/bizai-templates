Retail Product Description

Retail & E-commerce Product & Service Descriptions

Generate content for online or physical store

Use Cases

Use for listing

Not Suitable For

Not suitable for product quality inspection reports

Materials Needed

  • product complete
  • (for)
  • data
  • use
  • core selling points
  • and real

Prompt (Click to Copy)

[Note: This prompt template is in Chinese, designed for use with Chinese-language AI models.]

你是一位为多个电商品牌撰写过商品详情页的资深文案，熟悉淘宝、拼多多、抖音小店的商品描述规范。现在需要为一件商品撰写详情页描述文案。

请根据以下信息撰写：
商品名称：{商品名称}
材质/成分/面料：{材质或成分}
规格尺寸（含尺码表如适用）：{规格尺寸}
适用人群和场景：{适用人群}
核心卖点（最多3个）：{卖点}
与同类商品的差异点：{差异点}

输出要求：
- 字数120-180字
- 开头一句话点明这个商品「解决什么问题」或「适合什么场景」
- 材质描述要用消费者能感知的方式（如「摸起来像XX」「穿上去XX感觉」）
- 至少包含2个具体数字（克重、尺寸、含量等）
- 卖点之间要有逻辑递进，不要简单罗列
- 结尾用一句话总结「什么人在什么场景下选它」
- 语气专业但不冷淡，像一个懂行的朋友在推荐

禁止事项：
- 不使用「最」「第一」「独家」「全网最低」等绝对化用语
- 不编造检测报告或认证
- 不贬低竞品
- 不使用「买到就是赚到」等促销话术
- 不承诺使用效果（如「穿了显瘦10斤」）

Field Descriptions

FieldDescription
{商品名称}product complete (e.g. "纯棉圆领短袖T恤")
{材质或成分}and (e.g. "100%新疆长绒棉，220g克重")
{规格尺寸}scope and (e.g. "S-3XL，衣长65-75cm")
{适用人群}use and use (e.g. "25-40岁通勤人群，日常穿搭")
{卖点}1-3 (e.g. "不起球、预缩水、落肩设计")
{差异点}and product has not (e.g. "比普通棉T厚30%但透气性不减")

Output Format

100-150 character product description

Output Example

[Sample output in Chinese - this is a format reference for Chinese-language AI output] 通勤穿搭最怕的就是起球和变形，这件T恤用的是新疆长绒棉，面料克重220g，洗了不容易变薄。做了预缩水处理，按正常尺码买就行，不用担心缩水问题。版型是微宽松的落肩设计，肩线下落2cm，不挑身材。S到3XL都有，155-185cm都能找到合适的码。男女同款，基础色百搭，日常通勤或周末出门都能穿。（仅供格式参考，实际请根据你的商品信息生成）

The above is a format example only. Actual output should be generated based on your materials.
