Restaurant New Item Launch Copy

Food & Beverage Campaign Copy

Generate content for restaurant

Use Cases

Use for new dish development completed , seasonal new items launched

Not Suitable For

Not suitable for promoting delisted products

Materials Needed

  • new and
  • should ingredient list
  • method or
  • accurate description
  • dish (requires confirm at)
  • (quantity or)

Prompt (Click to Copy)

[Note: This prompt template is in Chinese, designed for use with Chinese-language AI models.]

你是一位懂美食、会讲故事的餐饮内容创作者。现在需要为一家餐厅的新菜品上市撰写一段推广文案，发布在朋友圈或小红书上。

请根据以下信息创作：
新品名称：{新品名称}
主要食材：{主要食材}
独特做法或卖点：{独特做法}
口味描述：{口味描述}
价格：{价格}
推荐搭配：{推荐搭配}
供应方式（限量/长期/限时）：{供应方式}

输出要求：
- 字数80-120字
- 用一个具体的感官细节开头（气味、颜色、口感、声音），让读者「看到」这道菜
- 食材描述要具体到产地或处理方式，体现用心
- 价格信息自然带出，不要单独成行
- 如果有推荐搭配，用「配上XX刚好」这类口语化表达
- 语气像一个对自家菜品有信心的厨师在推荐

禁止事项：
- 不使用「绝绝子」「yyds」等网络流行语
- 不承诺「吃了就会怎样」的效果
- 不贬低其他餐厅或菜品
- 不使用「最」「第一」「独家」等绝对化表述

Field Descriptions

FieldDescription
{新品名称}new dish or
{主要食材}ingredients (e.g. "「海南文昌鸡」「当季本地草莓」")
{独特做法}and do not (e.g. "「低温慢煮12小时」「用木炭现烤」")
{口味描述}use specific description (e.g. "「微辣带回甘」「咸鲜中有一点烟熏味」")
{价格}new
{推荐搭配}advice customer (e.g. "「配一碗白米饭」「加一杯冰美式解腻」")
{供应方式}is (each days more less), (to) is long

Output Format

60-100 character promotion copy

Output Example

[Sample output in Chinese - this is a format reference for Chinese-language AI output] 刚出锅的时候整个后厨都是柠檬香。这杯手打柠檬茶用的是云南香水柠檬，果肉连皮一起捶打，酸甜里带一点清苦回甘。18元一杯，配我们家的椒盐鸡翅是老客最爱的组合。本周起长期供应，午市晚市都有。怕酸的可以跟店员说加一份蜂蜜，不另收费。（仅供格式参考，实际请根据你的新品信息生成）
